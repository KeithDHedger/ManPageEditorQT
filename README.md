# ManPageEditorQT
A manpage editor for QT  
I now consider this usable, but please make backups when changing man pages.  


### Compile
```

For Qt6:
./autogen.sh --prefix=/usr

For Qt5:
USEQTVERSION=5 ./autogen.sh --prefix=/usr


```
